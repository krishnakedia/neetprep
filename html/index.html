Important Setup Instructions

1. Ensure Laravel is running at http://127.0.0.1:8000
2. Ensure MySQL database is running with the neetprep database created
3. Run migrations: php artisan migrate
4. Run seeders: php artisan db:seed (optional, creates demo data)
5. Start the Angular dev server: npm start (for live development)
6. These HTML files can be used standalone with the Laravel API backend
